JSON API仕様

パラメーター
format 出力フォーマット format=jsonを指定してください。指定しないとテスト用のHTML（このページ）が出力されます。
q 検索クエリーを文字列で指定。デフォルトは指定なし。指定しないまたは空文字だと最新情報が出力される。検索対象は県と市町村
page ページ番号を数値で指定。1から始まる。デフォルトは1
limit 出力件数を数値で指定。デフォルトは60
callback JSONPの関数名を指定可能。デフォルトは指定無し
status 状態（status）に対する検索クエリーを文字列で指定。デフォルトは指定無し。"yes"が「無事を確認できました」、"no"が「無事を確認できませんでした」のエイリアス。
link コメントの@usernameとURLをクリッカブルURLにするかどうか。デフォルトは指定無し。"yes"でリンクにする

サンプル
/?format=json&page=1&limit=5&status=yes
